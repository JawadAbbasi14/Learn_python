## ğŸ“ Python Basics

### 1. Hello World ğŸŒ
Your first Python program:
```python
print("Hello, World!")
```

### 2. Variables and Data Types
```python
# Integer
a = 10

# Float
b = 3.14

# String
c = "Python is fun!"

# Boolean
d = True
```

### 3. Conditional Statements
```python
x = 5
y = 10
if x < y:
    print("x is less than y")
else:
    print("x is greater than or equal to y")
```

### 4. Loops
#### For Loop:
```python
for i in range(5):
    print(i)
```
#### While Loop:
```python
count = 0
while count < 5:
    print(count)
    count += 1
```

### 5. Functions
```python
def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))
```
